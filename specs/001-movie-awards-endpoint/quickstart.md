# Quickstart: Movie Awards Endpoint

## Prerequisites

- Java 17 or later
- Maven
- Existing application running locally

## Validation Steps

1. Start the application with Maven.
2. Call the new endpoint through the recognition API.
3. Confirm the response includes the movie title, total awards, Oscar wins, and BAFTA wins.
4. Verify the returned movie is the one with the highest award total in the seeded data.
